# Sneak Peak LLC Website

A professional website for Sneak Peak LLC, a photography and imaging equipment business serving businesses, higher education, and religious organizations.

## Features

- Clean, professional design
- Responsive layout (mobile-friendly)
- Contact form for inquiries
- No e-commerce functionality (as requested)
- No product catalogue (as requested)

## Files

- `index.html` - Main HTML file
- `styles.css` - Styling and design
- `script.js` - JavaScript for form handling and interactions
- `README.md` - This file

## Setup

1. Open `index.html` in a web browser to view the site locally
2. Customize the content as needed
3. Update the email in `script.js` (line 20) with your actual email address

## Cheapest Hosting Solutions (FREE)

### Option 1: GitHub Pages (Recommended - 100% Free)
1. Create a GitHub account (free)
2. Create a new repository
3. Upload all files to the repository
4. Go to Settings â†’ Pages
5. Select main branch and save
6. Your site will be live at: `https://yourusername.github.io/repository-name`

**Pros:** Free forever, easy to update, reliable
**Cons:** Requires GitHub account

### Option 2: Netlify (Free Tier)
1. Create account at netlify.com (free)
2. Drag and drop your website folder
3. Site is live instantly
4. Free custom domain support

**Pros:** Very easy, instant deployment, free SSL
**Cons:** Limited bandwidth on free tier (usually enough for small sites)

### Option 3: Vercel (Free Tier)
1. Create account at vercel.com (free)
2. Import from GitHub or upload files
3. Automatic deployments

**Pros:** Fast, modern platform, great performance
**Cons:** Similar to Netlify

### Option 4: Cloudflare Pages (Free Tier)
1. Create account at cloudflare.com (free)
2. Connect GitHub or upload files
3. Free hosting with CDN

**Pros:** Excellent performance, free CDN, unlimited bandwidth
**Cons:** Slightly more technical setup

## Recommended: GitHub Pages

For the absolute cheapest solution (completely free), I recommend **GitHub Pages**. It's:
- 100% free forever
- Easy to set up
- Reliable and fast
- Easy to update (just push changes)

## Custom Domain (Optional)

All hosting platforms above support custom domains:
- GitHub Pages: Free custom domain support
- Netlify: Free custom domain + SSL
- Vercel: Free custom domain + SSL
- Cloudflare: Free custom domain + SSL

Domain costs: ~$10-15/year (Namecheap, Google Domains, etc.)

## Email Setup

The contact form currently uses a mailto link. For better functionality, consider:

1. **Formspree** (Free tier: 50 submissions/month)
   - Easy integration
   - Just add their endpoint to your form

2. **EmailJS** (Free tier: 200 emails/month)
   - Send emails directly from JavaScript
   - No backend needed

3. **Netlify Forms** (Free tier: 100 submissions/month)
   - If hosting on Netlify
   - Built-in form handling
